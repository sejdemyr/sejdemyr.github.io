R Tutorials: Statistics

By Simon Ejdemyr

These tutorials were originally developed for an undergraduate class in quantitative methods in the social sciences at Stanford.

  • Tutorial 1: Basic syntax, reading data, loading & installing packages, basic data manipulation.
  • Tutorial 2: More on data manipulation, graphing with ggplot, for loops, central limit theorem.
  • Tutorial 3: More on for loops, graphing with ggplot, functions, coverage probability.
  • Tutorial 4: Exporting tables & graphs, hypothesis testing.
  • Tutorial 5: Merging & appending data.
  • Tutorial 6: Interpreting & graphing OLS models.
  • Tutorial 7: Analyzing (quasi-)experimental data.
  • Tutorial 8: Propensity score matching.
